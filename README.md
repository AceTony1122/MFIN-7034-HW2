## How to download the notebook

**Option 1 – Direct download from GitHub**

1. Click [`HW2_Portfolio_Analysis.ipynb`](HW2_Portfolio_Analysis.ipynb) in the file list above (or at the top of this repo page).
2. On the file view page, click the **Download raw file** button (⬇ icon, top-right of the file viewer).

**Option 2 – Clone the repository**

```bash
git clone https://github.com/AceTony1122/MFIN-7034-HW2.git
cd MFIN-7034-HW2
jupyter notebook HW2_Portfolio_Analysis.ipynb
```
